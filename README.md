
# Bénin Insights Challenge — Team 2
### iSHEERO × DataCamp Donates | Hackathon 2026

## Objectif
Extraire et analyser les données GDELT des 12 derniers mois
concernant le Bénin, et en produire des insights utiles pour
les journalistes, chercheurs et décideurs publics.

## Stack technique
- **Données** : GDELT via Google BigQuery
- **Traitement** : Python, Pandas
- **Visualisation** : Streamlit, Plotly
- **ML** : Scikit-learn, HuggingFace
- **Collaboration** : GitHub
