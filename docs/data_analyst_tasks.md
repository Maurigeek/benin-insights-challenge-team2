# Tâches Data Analyst

### DA-01 — Explorer les données
- Lire `tone_monthly.csv`, `events_by_type.csv`, `actors_country.csv`, `geo_events.csv`.
- Identifier les variables utiles, la période couverte et les valeurs manquantes.
- Repérer les 3 à 5 tendances majeures et les anomalies visibles.
- Livrable : note courte d’exploration avec points clés.

### DA-02 — Faire les visuels
- Produire les graphiques nécessaires au notebook et au dashboard.
- Cibler au minimum : tendance temporelle, répartition par type d’événement, acteurs/pays, carte ou géographie.
- Vérifier lisibilité, titres, axes, légendes et cohérence des couleurs.
- Livrable : jeu de visuels prêt à intégrer.

### DA-03 — Sortir les insights
- Rédiger 5 à 7 insights maximum.
- Chaque insight doit être relié à une donnée ou un graphique précis.
- Prioriser : faits marquants, variations fortes, signaux utiles pour le jury.
- Livrable : liste d’insights courts, vérifiables et exploitables.

### DA-04 — Préparer l’analyse finale
- Organiser l’analyse en : contexte, constats, lecture des tendances, conclusion.
- Vérifier que le texte correspond exactement aux visuels utilisés.
- Livrable : bloc d’analyse prêt pour le notebook / rapport.
