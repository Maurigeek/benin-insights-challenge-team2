# Tâches Data Scientist

### DS-01 — Définir la problématique
- Clarifier la question à traiter.
- Définir le périmètre d’analyse.
- Livrable : cadrage clair du sujet.

### DS-02 — Analyser les tendances
- Étudier les tendances majeures issues des données.
- Isoler les signaux importants.
- Livrable : tendances clés documentées.

### DS-03 — Interpréter les insights
- Relier les insights aux données et graphiques.
- Prioriser les constats les plus utiles.
- Livrable : insights interprétés et argumentés.

### DS-04 — Rédiger la synthèse finale
- Structurer la synthèse pour le rapport/notebook.
- Vérifier la cohérence entre constats et preuves.
- Livrable : synthèse finale prête à intégrer.
