# Tâches ML Engineer

### ML-01 — Préparer le dataset ML
- Construire le dataset d’entraînement à partir des données processed.
- Vérifier les colonnes d’entrée et la cible.
- Livrable : dataset ML prêt à entraîner.

### ML-02 — Construire le modèle v1
- Tester un premier pipeline de modélisation.
- Entraîner au moins un modèle baseline.
- Livrable : modèle v1 entraîné.

### ML-03 — Évaluer les performances
- Mesurer les métriques principales.
- Comparer les résultats entre versions/tests.
- Livrable : résultats d’évaluation clairs.

### ML-04 — Améliorer rapidement le modèle
- Ajuster les features et paramètres principaux.
- Retenir la version la plus robuste.
- Livrable : modèle v1 amélioré.

### ML-05 — Préparer l’intégration
- Exporter les sorties utiles pour dashboard/rapport.
- Documenter comment relancer l’entraînement et la prédiction.
- Livrable : artefacts ML utilisables par l’équipe.
